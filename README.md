# Life English｜AI人生英语

Life English是一款以真实生活任务为主线的英语学习网页APP。学习者可以从A1—A2、B1—B2或C1路线开始，在酒店、机场、合租、面试与职场协商等情境中完成表达任务，并通过AI对话、互动短剧、词库复习和成长记录形成学习闭环。

## 在线体验

## 核心模块

- 分层情境地图与路线解锁
- 选择题对话与即时反馈
- AI自由对话实验室
- 互动短剧、原声与跟读练习
- 真实表达词库与AI复习
- XP、打卡、勋章与成长记录
